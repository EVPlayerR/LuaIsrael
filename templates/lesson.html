הדפסה

הקדמה:

פרינט זו פקודה המדפיסה מה שרשום בתוכה. בשביל להשתמש בפרינט נצטרך לרשום את הקוד הבא:

print("Hello World")

תוצאה:

Hello World

כלומר, שזה ידפיס את הערך או הטקסט שבתוך הסוגריים.

למה זה יעיל?

הפקודה print משמשת להדפסת טקסט או ערך.
זה כלי יעיל לאיתור בעיות בקוד ולהבנת תהליכי הרצה. כלומר, כשיש בעיה, הפלט(התגובה) שנוצר

קלט / פלט

הקדמה:

האווטפוט (Output) זה הפלט. הכוונה היא התוצאה של הקוד, לכל קוד יש Input ויש גם Output. הInput הוא הקלט (הקוד עצמו)

דוגמה לInput:

Input --

print("Hello World")

והתוצאה היא הOutput:

Output --
Hello World

בוליאנים

הקדמה:

בוליאנים (booleans) הם נכון או לא נכון (True / False)

דוגמה --

script.Parent.Enabled = True

כלומר, שזה יעשה את מה שנמצא בscript.parent מופעל.

למה זה יעיל?

בכדי להגדיר ערכים / משתנים כTrue או False, ולכבות או להפעיל דברים, לעשות שכאשר משהו לא נכון אז יקרה משהו, וכו...

נכון או לא נכון --

True = נכון
False = לא נכון


לולאות

הקדמה:

ישנם כמה סוגים של לולאות, לולאה גורמת לשורת קוד או לתוכנית שלמה לחזור על עצמה מספר פעמים, יכול להיות שהלולאה תימשך לנצח ויכול להיות שהיא תעצר בשלב מסויים.

while true do
print("Hello World")
end

קוד זה ידפיס "Hello World" לנצח ללא הפסקה, ניתן לגרום להפסקה של מספר שניות בלולאה בצורה הבאה:

while wait(5) do
print("Hello World")
end

וגם ניתן ליצור לולאה שתמשיך עד פעולה מסויימת:

local a = 1

repeat
wait(0.1)
a = a + 1
until a == 5

בדוגמה זו הגדרנו שa שווה 1, ויצרנו לולאה שכל 0.1 שניות זה יעלה את a באחד, וזה יעצור את הלולאה כאשר a יהיה שווה 5.

למה זה יעיל?

לולאה משמשת לחזרה על קוד שוב ושוב, או עד פעולה מסויימת. עם זה אפשר ליצור כל מיני דברים, לדוגמה: סקריפט שכל מספר שניות מביא מספר כסף מסויים לשחקן.

הצהרות

הקדמה:

הצהרות הן מה יקרה אם, או אחרת. זאת אומרת להפעיל פעולה מסויימת אם קורה משהו מסויים, אם תנאי מתקיים.
יש כמה סוגי הצהרות:

if --
a = 5
if a == 5 then
print("True")

בעזרת else ניתן לבצע פעולה ב"כל מקרה אחר" אם התנאי לא מתקיים:

else --
a = 4
if a == 5 then
print("True")
else
print("False")
end

וגם ניתן לבצע בדיקה של תנאי חדש במידה והתנאי הקודם לא מתקיים:

elseif --
a = 5
if a == 2 then
print("a = 2")
elseif a == 5 then
print("a = 5")
end

למה זה יעיל?

ניתן להשתמש בהצהרות אם אנחנו רוצים לבצע פעולה מסויימת אם תנאי מסויים מתקיים, או אם הוא לא מתקיים.

משתנים

הקדמה:

משתנים הם כלי חשוב בתכנות ברובלוקס סטודיו, שיעזרו לך לארגן ולשדרג את הקוד שלך. במדריך הזה, נלמד כיצד להשתמש במשתנים וליצור אותם בצורה יעילה.

השימוש במשנים:

הגדרת משתנה --
a = "Hello"

הדפסת השתנה --print(a)

בדוגמה הזו, המשתנה a הוגדר כ־"Hello", ואז נדפיס אותו. התוצאה תהיה "Hello". מטרת המשתנים היא להקל עליך ולהפוך את הקוד לקריא ומובן יותר.

יצירת משתנים:

יצירת משתנה חדש --
a = "World"

למה זה יעיל?

משתנים יעילים על מנת לקצר את הזמן, במקום לכתוב הרבה, להגדיר משתנה קצר.

הקדמה:

צ'אט עברית מאפשר לשחקנים במשחק שלך לדבר חופשי בשפה העברית, הצ'אט שלנו מאפשר זאת.

איך להוריד את הקובץ?

בשביל להוריד את הקובץ של הצ'אט עברית אתה יכול להכנס לשרת הדיסקורד שלנו או ללחוץ על הכפתור למטה.
לחץ כאן להורדת הקובץ
